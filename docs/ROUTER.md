# ROUTER MANAGEMENT 

Routes are on src/ezway/Config/Routes.php

## CREATE A WEB ROUTE

Add a entry in array : ['method' => 'GET', 'path' => '/example', 'callback' => 'Example.show'].  
*path* : route name  
*callback* : [controller name].[function name]
